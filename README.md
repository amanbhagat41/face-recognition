## About The Project

<!-- [![Product Name Screen Shot][product-screenshot]](https://example.com) -->

This project was made during a web developer online class I took on Udemdy. This website is a fullstack website that has a log-in and signup page that is stored into a database. Once you login you will see a box to insert an image url and once it is entered, the website will let you know how many faces are in thay photo as well as outline the faces in a box.

## Getting Started

To set up the project just download the code files and get an Api key from clarifai and insert it into the code and run

### Prerequisites

This is an example of how to list things you need to use the software and how to install them.
* npm
  ```sh
  npm install npm@latest -g
  ```

### Installation

1. Get a free API Key at [https://www.clarifai.com/](https://clarifai.com/clarifai/main/models/face-detection)
2. Clone the repo
   ```sh
   git clone https://github.com/amanbhagat41/face-recognition.git
   ```
3. Install NPM packages
   ```sh
   npm install
   ```
4. Enter your API in `config.js`
   ```js
   const API_KEY = 'ENTER YOUR API';
   ```
